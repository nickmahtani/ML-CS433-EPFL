### Task C

- Implement the assignment of data to two given distributions
- **Determinant** is a single number that tells you how much a matrix spreads space
    - Big determinant: data is spread out widely
    - Small determinant: data is tight, concentrated
    - Measures size/area of the spread
- **Covariance matrix** defines the shape and orientation of the spread
    - Captures how the variance changes in different directions
    - Interdimensional dependencies for variance
    - The diagonals of the covariance matrix are the variances
- **Density model** is a formula that gives the likelihood (how well a model explains a point) — describes how data is spread
